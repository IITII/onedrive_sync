# onedrive 分享链接下载/同步工具

* 将 onedrive 的文件下载/同步到本地
* 支持增量同步, 同路径为空文件自动跳过, 删除即可重新同步
* 支持多线程下载, 速度快

* 目前只支持不带密码的分享链接
* 默认只下载图片, 其他文件类型默认跳过
* 配置详见 [config.js](./config/config.js)
